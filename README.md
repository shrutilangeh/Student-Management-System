# Student Management System

This is a CLI-Based Java CRUD Application created using Core Java concepts.

## Features
- Add Student
- View Students
- Update Student
- Delete Student

## Concepts Used
- Classes and Objects
- ArrayList
- Scanner
- Methods
- Loops and Conditionals

## Tools Used
- Java
- VS Code

## How to Run
1. Open the project in VS Code
2. Run the StudentManagementSystem.java file
3. Use menu options to perform CRUD operations
